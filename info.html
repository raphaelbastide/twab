Why this is is cool?

  • Light
  • Self-hosted
  • Radically raw
  • Browser-based
  • This is universal
  • Highly customisable
  • Finger protocol compatible
  • The HTML page’s content can be anything
  • The tab’s favicon can be used as personal picture…
